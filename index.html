Downloading

Simply click de "Download" button in the navbar or the button below and you will get the CSS, JS and font files.

Download

Integrate files

You need to integrate the CSS files into the HEAD of your HTML. Furthermore a "viewport" is needed, to make sure the website functions correctly on smartphones with a high screen resolution.
<link href="source/jada.min.css" rel="stylesheet">
<meta name="viewport" content="width=device-width, initial-scale=1">
To ensure the website is loaded quickly, integrate the JS files at the bottom of your BODY. Download the JQuery file or integrate it with the direct link.
<script src="source/jada.min.js"> </script>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/2.2.4/jquery.min.js"> </script>
